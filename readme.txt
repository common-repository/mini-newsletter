== Description ==
Mini Newsletter, where you can send a custom email as well as the latest post of your blog to your email subscribeer. You can send both custom email as well as latest post to your subscribers.  

1. Send custom email to your subscriber or anyone.
2. Use shortcode ( [newsletter_form] ) to show the subscriber form. 
3. Delete existing subscriber from the email list.


== Installation ==
1. Visit Plugins > Add New
2. Search for “Mini Newsletter”
3. Install the “Mini Newsletter” plugin
4. Activate "Mini Newsletter" from your Plugins page
5. Use [newsletter_form] shortcode to show the subscriber form.


== Frequently Asked Questions ==
= How many latest post I can email?
You can email upto 10 latest post to your subscriber
= Is there any email template?
Yes, current there are 1 email template. More will come soon. 
= Can I send custom email to anyone?
Yes, you can send custom email to anyone you want.
= Is there any way to change the email header name and email address?
Yes, Go to Settings tab and find the Email header name and email input box.
